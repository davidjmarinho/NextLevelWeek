npm init -y (Serve pro npm nao fazer perguntas chatas e definir tudo padrao)

npm install express (Serve para gerenciar as rotas)

npm install @types/express -D (Para instalar a lib de definicao de tipos do express)

npm install typescript -D (Instala o modulo do typescript)

npm install ts-node -D (instala o executor do typescript com o node)

npx tsc --init (Cria um arquivo de config do typescript  )

npx ts-node src/server.ts (Executa o arquivo typescript)

npm install ts-node-dev -D (O pacote para compilar automaticamento)
